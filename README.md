# Starbucks Corporation (SBUX)
# Опис ідеї та обґрунтування вибору фінансового інструменту:
Ми обрали акції Starbucks, оскільки це одна з найбільших міжнародних мереж кав'ярень із багаторічною історією та стабільною присутністю на фондовому ринку. На відміну від популярних прикладів із технологічними компаніями чи криптовалютами, Starbucks представляє сферу споживчих товарів і послуг. Дані доступні, що робить розрахунки коректнішими. Нам було цікаво перевірити, чи буде стратегія перетину ковзних середніх ефективною.
# Опис процесу аналізу (які інструменти та бібліотеки використовувалися):
yfinance - для завантаження історичних ринкових даних (Open, High, Low, Close, Volume) з Yahoo Finance,
pandas - для структуризація даних у DataFrame та проведення розрахунків ковзних середніх і торгових сигналів,
matplotlib - для візуалізації результатів (графіки ціни, ліній SMA та точок купівлі/продажу)
# Короткі висновки щодо ефективності вашої торгової стратегії:
Cтратегія на основі перетину ковзних середніх успішно генерувала сигнали купівлі та продажу. За історичними даними Starbucks за 2025 рік показав додатній фінансовий результат - приблизно 19.45, що свідчить про його ефективність на вибраному періоді.
