# Starbucks Corporation (SBUX)
 Опис ідеї та обґрунтування вибору фінансового інструменту:
Ми обрали акції Starbucks, оскільки це одна з найбільших міжнародних мереж кав'ярень із багаторічною історією та стабільною присутністю на фондовому ринку. На відміну від популярних прикладів із технологічними компаніями чи криптовалютами, Starbucks представляє сферу споживчих товарів і послуг. Дані доступні, що робить розрахунки коректнішими. Нам було цікаво перевірити, чи буде проста стратегія перетину ковзних середніх ефективною для акцій компанії, попит на продукцію якої формується щоденними звичками споживачів.
 Опис процесу аналізу (які інструменти та бібліотеки використовувалися):
Python - мова програмування,
yfinance - для завантаження історичних ринкових даних (Open, High, Low, Close, Volume) з Yahoo Finance,
pandas - для структуризація даних у DataFrame та проведення векторизованих розрахунків ковзних середніх і торгових сигналів,
matplotlib - для візуалізації результатів (графіки ціни, ліній SMA та точок входження/виходу у позицію),
